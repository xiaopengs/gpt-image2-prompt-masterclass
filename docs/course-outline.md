## 第二篇：入门篇 —— 提示词基础与美学感知（约 12,000 字）

### 第2章：提示词到底是什么？—— 万能公式入门

**学习目标：** 掌握提示词的基本结构与万能公式

**核心知识点：**
- 提示词万能公式：`主体 + 场景 + 风格 + 光线 + 构图 + 细节修饰`
- 六大要素逐一拆解：
  - **主体（Subject）：** 人物/物体/场景的核心描述
  - **场景（Setting）：** 环境与背景
  - **风格（Style）：** 艺术风格与视觉调性
  - **光线（Lighting）：** 光源类型与氛围
  - **构图（Composition）：** 镜头语言与画面布局
  - **细节修饰（Details）：** 材质、纹理、情绪等微调
- 简短提示词 vs 长提示词的适用场景

**实战练习：** 用万能公式写 5 条基础提示词并生成图片

**案例分析：** 对比仓库中 Case 2（极简电影人像，短提示词）与 Case 1（便利店霓虹人像，长提示词），分析结构差异

### 第3章：美学描述入门 —— 学会用"画面感"说话

**学习目标：** 建立美学词汇库，学会描述视觉感受

**核心知识点：**
- 颜色与色调词汇：warm tones / cool ivory / neon glow / pastel / high contrast
- 氛围词汇：cinematic / intimate / dreamy / moody / ethereal
- 质感词汇：film grain / glossy / matte / soft focus / sharp detail
- 构图词汇：medium shot / close-up / bird's eye view / symmetrical / dynamic angle
- 从"好看"到"精确描述"的转化练习

**实战练习：** 给 5 张图片写出完整的美学描述

**案例分析：** 拆解 Case 3（日式温泉旅馆人像）中的美学描述层次

### 第4章：专业词汇速查 —— 摄影与设计必学术语

**学习目标：** 掌握提示词中高频出现的专业术语

**核心知识点：**
- **摄影术语：** 35mm film、DSLR、depth of field、bokeh、golden hour、harsh lighting、soft box
- **胶片与相机：** Fujifilm、CCD camera、black mist filter、film grain、color grading
- **设计术语：** minimalist、typography、layout、infographic、flat design、glassmorphism
- **风格术语：** editorial、cinematic、vintage、retro、cyberpunk、wuxia、ink wash
- **技术参数：** aspect ratio（16:9、9:16、1:1）、equirectangular（360°全景）

**实战练习：** 完成专业术语匹配游戏 + 用 10 个新学术语写提示词

**案例分析：** 对比仓库中同一主题用不同术语描述的效果差异
